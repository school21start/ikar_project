# Система машинного зрения для автоматизации контроля дефектов штамповки

Проект разработан совместно с ПАО «Челябинский кузнечно-прессовый завод» (ЧКПЗ) и ИТ-кампусом «Школа 21» в рамках конкурса «Инженерные кадры России 2025–2026».

Система предназначена для автоматического обнаружения дефектов горячей штамповки с использованием технологий компьютерного зрения и нейронных сетей YOLOv8.

---

## О проекте

На производстве ПАО «ЧКПЗ» выходной контроль качества изделий выполняется вручную, что приводит к рискам пропуска дефектов из-за человеческого фактора.

Проект решает задачу автоматизации контроля качества изделий после горячей штамповки:

- обнаружение дефектов в реальном времени;
- снижение количества пропущенного брака;
- круглосуточная работа без потери качества контроля;
- возможность интеграции с промышленной линией.

---

## Цель проекта

Разработка системы машинного зрения на базе YOLOv8 для автоматического обнаружения дефектов штампованных изделий:

- незаполнение угловых полостей;
- облой.

## Архитектура системы

Система работает в двух режимах:

### 1. Standby Mode («Ждущий режим»)

Камера анализирует изменения в кадре с помощью OpenCV MOG2.

Если движение отсутствует:

- GPU не используется;
- нейросеть не запускается;
- снижается энергопотребление.

### 2. Active Mode («Режим контроля»)

При появлении изделия:

1. кадр передаётся в YOLOv8;
2. выполняется поиск дефектов;
3. формируется результат инференса;
4. система отправляет сигнал:
   - оператору;
   - световому маяку;
   - механизму отбраковки.
